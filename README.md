# 🎨 Background Color Input

## 📌 Sobre o projeto

O Background Color Input é um projeto simples e interativo que permite ao usuário alterar a cor de fundo da página em tempo real utilizando um seletor de cores.

Este projeto foi desenvolvido com o objetivo de praticar manipulação do DOM e interatividade com JavaScript.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

---

## 🎯 Objetivo

Este projeto foi criado para:

- Praticar lógica de programação com JavaScript
- Aprender manipulação do DOM
- Trabalhar com eventos de entrada do usuário
- Melhorar minhas habilidades em desenvolvimento front-end

---

## ⚙️ Funcionalidades

- Selecionar uma cor usando um input
- Alterar a cor de fundo dinamicamente
- Interface simples e intuitiva
